# 10 - 高级算法模块需求文档

## 1. 模块概述

高级算法是计算机科学的核心支柱，也是技术面试中区分候选人的关键考察点。本模块涵盖动态规划、贪心算法、回溯法、分治法和位运算技巧五大子模块，这些算法范式不仅是解决复杂问题的利器，更是培养算法思维和系统设计能力的基石。

当前项目已实现基础排序算法（冒泡排序、快速排序、插入排序、希尔排序、选择排序、堆排序、归并排序）和基础查找算法（线性查找、折半查找），但高级算法范式完全缺失。本模块将从零构建，为项目补齐这一关键短板。

**目标定位：**
- 系统掌握五大算法范式的核心思想与适用场景
- 能够独立分析问题并选择合适的算法策略
- 覆盖 LeetCode 高频面试题的核心题型
- 建立从暴力解法到最优解法的优化思维路径

## 2. 缺失内容

本模块为全新模块，所有内容均需从零构建：

| 子模块 | 当前状态 | 优先级 |
|--------|---------|--------|
| 动态规划 | ❌ 完全缺失 | P0 - 最高 |
| 贪心算法 | ❌ 完全缺失 | P1 - 高 |
| 回溯法 | ❌ 完全缺失 | P1 - 高 |
| 分治法 | ❌ 完全缺失 | P2 - 中（部分算法已在排序模块中实现） |
| 位运算技巧 | ❌ 完全缺失 | P2 - 中 |

#### 3.1.1 核心概念

| 概念 | 说明 | 重点程度 |
|------|------|---------|
| 最优子结构 | 问题的最优解包含子问题的最优解 | ⭐⭐⭐ |

| 状态转移方程 | 描述问题各阶段之间关系的递推公式 | ⭐⭐⭐ |

| 自顶向下（Memoization） | 从原问题出发递归求解，记忆化搜索 | ⭐⭐ |

#### 3.3.3 实现要求

- 每个问题需绘制状态空间树示意图（至少前 3 层）
- 明确标注剪枝条件及其效果（减少搜索节点数的估算）
- 提供 N 皇后的多种解法对比：基础回溯 → 对角线数组优化 → 位运算优化
- 回溯模板需统一：

```cpp
void backtrack(状态参数) {
    if (终止条件) {
        收集结果;
        return;
    }
    for (选择 : 本层所有选择) {
        做选择;
        backtrack(更新后的状态参数);
        撤销选择;
    }
}
```

### 3.5 位运算技巧（Bit Manipulation）

#### 3.5.1 常用位运算操作

| 操作 | 表达式 | 说明 |
|------|--------|------|
| 获取第 i 位 | `(n >> i) & 1` | 判断第 i 位是否为 1 |
| 置位第 i 位 | `n \| (1 << i)` | 将第 i 位设为 1 |
| 清除第 i 位 | `n & ~(1 << i)` | 将第 i 位设为 0 |
| 翻转第 i 位 | `n ^ (1 << i)` | 将第 i 位取反 |
| 最低位 1 | `n & (-n)` | 提取最低位的 1（Brian Kernighan） |
| 清除最低位 1 | `n & (n - 1)` | 去掉最低位的 1 |
| 判断 2 的幂 | `n > 0 && (n & (n-1)) == 0` | 2 的幂只有一个 1 |

| 序号 | 问题 | LeetCode 题号 | 面试频率 | 难度 | 关键知识点 |

| 1 | 只出现一次的数字 | LC 136 | ⭐⭐⭐ | 简单 | 异或性质：a^a=0, a^0=a |
| 2 | 比特位计数 | LC 338 | ⭐⭐ | 简单 | DP + 位运算：dp[i] = dp[i>>1] + (i&1) |
| 3 | 2 的幂 | LC 231 | ⭐⭐ | 简单 | n & (n-1) 技巧 |
| 4 | 颠倒二进制位 | LC 190 | ⭐⭐ | 简单 | 逐位翻转、分治法（查表法） |

## 6. 学习路径

### 6.1 阶段一：基础建立（2-3 周）

```
位运算技巧 → 斐波那契数列 → 爬楼梯 → 最大子数组和
```

- 目标：建立 DP 基本思维，掌握状态定义和转移方程推导
- 位运算作为独立技能点，可并行学习

### 6.2 阶段二：核心突破（3-4 周）

```
0-1 背包 → 完全背包 → 零钱兑换 → 最长递增子序列 → 最长公共子序列
```

- 目标：掌握背包问题体系，理解一维/二维 DP 的区别与联系
- LIS 需同时掌握 O(n²) 和 O(n log n) 两种解法

### 6.3 阶段三：策略拓展（3-4 周）

```
回溯法基础（全排列/组合）→ N 皇后 → 子集和 → 贪心算法（活动选择/区间调度）
```

- 目标：掌握回溯法模板和剪枝技巧，理解贪心策略的适用条件
- 回溯与贪心可交替学习，对比"探索所有可能"与"局部最优"的思维差异

### 6.4 阶段四：综合应用（2-3 周）

```
编辑距离 → 打家劫舍系列 → 矩阵链乘法 → 分治法（最大子数组分治解法/最近点对）
```

- 目标：处理复杂状态转移，掌握区间 DP、状态机 DP 等高级技巧
- 分治法侧重理论分析，与已实现的排序算法建立联系

### 6.5 阶段五：面试冲刺（持续）

- 按面试频率标注（⭐⭐⭐）的问题进行重点练习
- 每道题需达到 15 分钟内写出正确解法的水平
- 定期回顾，保持手感

| 0-1 背包 | 恰好装满 vs 最多装满、方案数 | 一维空间优化时逆序遍历 |
| 零钱兑换 | 最少硬币数、方案数、组合 vs 排列 | 初始值设置（dp[0]=0，其余=INF） |
| 全排列 | 含重复元素、字典序 | used 数组 vs 交换法的选择 |
| 组合 | 剪枝优化、去重 | 排序后相邻元素去重 |
| 活动选择/区间调度 | 会议室问题变体 | 按结束时间排序而非开始时间 |
| 编辑距离 | 空间优化到 O(min(m,n)) | 边界条件、操作定义 |
| 汉明距离 | 位运算实现 | 异或后逐位计数 |

### 7.2 建议掌握（⭐⭐ 中频面试题）

| 问题 | 说明 |
|------|------|
| N 皇后 | 展示回溯法能力，位运算优化为加分项 |
| 打家劫舍 | 状态机 DP 的典型应用 |
| 最长公共子序列 | 二维 DP 基础题 |
| 分数背包 | 与 0-1 背包对比 |
| 子集和 | 回溯+剪枝的综合应用 |
| 只出现一次的数字 | 位运算入门 |

### 7.3 了解即可（⭐ 低频/进阶题）

| 问题 | 说明 |
|------|------|
| 矩阵链乘法 | 区间 DP 思想，面试极少直接考 |
| Strassen 矩阵乘法 | 理论意义大于实践，了解分治优化思路 |
| 最近点对 | 几何分治经典问题，了解算法思想 |
| 数独求解 | 综合性回溯题，面试较少出现 |
| 图着色 | NP 完全问题，了解回溯解法 |

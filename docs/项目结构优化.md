### 2.1 命名规范

| 规则 | 说明 | 示例 |
|------|------|------|
| 目录名使用全小写英文 | 使用 snake_case | `sequential_list/` 而非 `Sequential_list顺序列表/` |
| 源文件名使用全小写英文 | 使用 snake_case | `sequential_list.cpp` 而非 `1_0_Sequential_list.cpp` |
| 头文件名使用全小写英文 | 使用 snake_case | `sq_list.h` 而非 `BinT.h` |
| 禁止中文字符出现在路径中 | 确保跨平台兼容性 | 禁止 `Stack栈/`、`树的增删改查.cpp` |
| 每个模块一个目录 | 便于扩展和测试 | `src/stack/sequential_stack/` |

### 5.2 时效性注意事项

- **C 标准**：当前最新为 ISO/IEC 9899:2018（C17），下一版 C23 预计 2025 年发布。建议项目以 C17 为基准，C11 为最低要求。参考：[ISO/IEC 9899 标准文档](https://www.iso.org/standard/74528.html)
- **C++ 标准**：当前最新为 ISO/IEC 14882:2020（C++20），C++23 已定稿。建议项目以 C++17 为基准。参考：[cppreference.com - C++ 标准文档](https://en.cppreference.com/w/cpp/chrono)
- **Python 版本**：Python 3.12+ 已移除 `imp` 模块，3.13+ 引入自由线程模式。建议最低支持 Python 3.10。参考：[Python 官方文档 - What's New](https://docs.python.org/3/whatsnew/)

# 快速开始

5 分钟上手数据结构与算法学习项目。

## 环境准备

### C/C++ 环境

| 平台 | 编译器 | 安装方式 |
|------|--------|---------|
| Windows | MSVC | 安装 [Visual Studio](https://visualstudio.microsoft.com/)（勾选"使用 C++ 的桌面开发"） |
| macOS | Clang | `xcode-select --install` |
| Linux | GCC | `sudo apt install build-essential cmake` |

### Python 环境

| 要求 | 版本 |
|------|------|
| Python | ≥ 3.10 |
| 第三方依赖 | networkx, matplotlib（仅可视化脚本需要） |

```bash
pip install -r python/requirements.txt
```

## 编译运行第一个程序

```bash
cmake -B build
cmake --build build --target 01_sequential_list_v1
./build/src/linear_list/sequential_list/01_sequential_list_v1   # Linux/macOS
build\src\linear_list\sequential_list\01_sequential_list_v1.exe  # Windows
```

## 运行 Python 算法测试

```bash
cd python
python -m algorithms.run_all
```

预期输出：`结果: 74 通过, 0 失败 / 共 74 个模块`

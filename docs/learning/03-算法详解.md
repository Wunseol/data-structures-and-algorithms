### 插入排序

#### 概念说明

插入排序的基本思想是将待排序序列分为已排序和未排序两部分，每次从未排序部分取出一个元素，在已排序部分中找到合适的位置插入。项目实现了两种变体：直接插入排序和折半插入排序。

> | Python | [L02_insertion_sort.py](../../python/algorithms/sorting/L02_insertion_sort.py#L10) | `insertion_sort` (L10) |

#### 权威参考来源

- 《算法导论》（Introduction to Algorithms, CLRS）第 2 章 "Getting Started"，插入排序作为分治思想的入门示例
- Donald E. Knuth, *The Art of Computer Programming*, Vol.3: Sorting and Searching, 第 5.2.1 节，详细分析了插入排序的比较与移动次数

#### 学习前置知识

- 数组的基本操作
- 顺序查找与折半查找
- 时间复杂度 O 符号表示法

#### 实现原理

- **直接插入排序**：从第 2 个元素开始，依次将每个元素插入到前面已排序序列的正确位置。使用顺序查找确定插入位置，同时后移大于待插入元素的记录。
- **折半插入排序**：在直接插入排序的基础上，利用折半查找确定插入位置，减少比较次数，但移动次数不变。

```cpp
void InsertSort(SqList &L)
{
	int i,j;
	for(i=2;i<=L.length ;i++)
	{
		if(L.r[i].key <L.r[i-1].key )
		{
			L.r[0]=L.r[i];
			for(j=i-1;L.r[j].key >L.r[0].key ;j--)
			{
				L.r[j+1]=L.r[j];
			}
			L.r[j+1]=L.r [0];
		}
	}
}
```

👉 [折半插入排序完整源码](../../src/sorting/02_sequential_list_sorting.cpp)

```cpp
void BInsertSort(SqList &L)
{
	int i,j,low,mid,high;
	for(i=2;i<=L.length ;i++)
	{
		if(L.r[i].key <L.r[i-1].key )
		{
			L.r[0]=L.r[i];
			low=1;
			high=i-1;
			while(low<=high)
			{
				mid=(low+high)/2;
				if(L.r[0].key <L.r[mid].key ) high=mid-1;
				else low=mid+1;
			}
			for(j=i-1 ;j>=high+1;j--)
			{
				L.r [j+1]=L.r[j];
			}
			L.r[high+1]=L.r[0];
		}
	}
}
```

#### 时间/空间复杂度分析

| 指标 | 直接插入排序 | 折半插入排序 |
|------|-------------|-------------|
| 平均时间复杂度 | O(n²) | O(n²) |
| 最坏时间复杂度 | O(n²) | O(n²) |
| 最好时间复杂度 | O(n) | O(n) |
| 空间复杂度 | O(1) | O(1) |
| 稳定性 | 稳定 | 稳定 |

折半插入排序的比较次数为 O(n log n)，但移动次数仍为 O(n²)，因此总体时间复杂度仍为 O(n²)。

#### 与 C++ STL 标准库实现的对比

| 维度 | 本项目实现 | C++ STL |
|------|-----------|---------|
| 算法 | 纯插入排序 | `std::sort` 使用 IntroSort（快排+堆排+插入排序混合），当子数组长度 ≤ 16 时切换到插入排序 |
| 数据结构 | 顺序表（1-indexed，带哨兵 `r[0]`） | 随机访问迭代器（0-indexed） |
| 哨兵机制 | 使用 `r[0]` 作为哨兵，避免内循环越界判断 | 无哨兵，依赖迭代器比较 |
| 适用场景 | 教学演示 | 工程生产：`std::sort` 在小规模子数组上使用插入排序作为优化 |

#### 已知变体和演进

- **二路插入排序**：利用循环缓冲区减少移动次数
- **表插入排序**：使用链表结构，移动次数降为 0，但比较次数不变
- **希尔排序**：插入排序的分组改进版（见下文）
- **库排序（Library Sort）**：GAP 插入排序，在数组中预留空位，时间复杂度 O(n log n)

#### 版本/平台相关注意事项

- 本项目使用 1-indexed 顺序表（`SqList`），`r[0]` 作为哨兵单元，不存储有效数据
- C++ 标准未规定 `std::sort` 的具体算法，但主流实现（GCC libstdc++、LLVM libc++、MSVC STL）均采用 IntroSort
- 插入排序在近乎有序的数据上接近 O(n)，常被用作高级排序的子过程

#### Python 实现

👉 [Python 实现](../../python/algorithms/sorting/L02_insertion_sort.py)

```python
def insertion_sort(arr: list[int]) -> list[int]:
    a = arr[:]
    n = len(a)
    for i in range(1, n):
        if a[i] < a[i - 1]:
            temp = a[i]
            j = i - 1
            while j >= 0 and a[j] > temp:
                a[j + 1] = a[j]
                j -= 1
            a[j + 1] = temp
    return a
```

**与 C/C++ 实现差异**：Python 版本使用 0-indexed 列表和切片复制，无需哨兵单元，函数返回新列表而非原地修改。

### 希尔排序

#### 概念说明

希尔排序又称"缩小增量排序"，是直接插入排序的改进版。通过设置增量序列，将序列分成若干子序列分别进行插入排序，随着增量逐步缩小，序列趋于有序，最终增量为 1 时做一次直接插入排序即可。

>
> | 语言 | 文件 | 关键函数/类 |

> | Python | [L02_shell_sort.py](../../python/algorithms/sorting/L02_shell_sort.py#L10) | `shell_sort` (L10) |

#### 权威参考来源

- Donald L. Shell, "A High-Speed Sorting Procedure", *Communications of the ACM*, Vol.2, No.7, 1959
- Knuth, *TAOCP* Vol.3, 第 5.2.1 节，分析了不同增量序列的性能
- Sedgewick, R., "Analysis of Shellsort and Related Algorithms", *ESA*, 1996 — 提出了目前实践中最优的 Sedgewick 增量序列

#### 学习前置知识

- 直接插入排序
- 增量/步长的概念
- 序列渐进有序性

#### 实现原理

1. 选择一个增量序列（本项目使用 {5, 3, 1}）
2. 对每个增量 dk，将序列按 dk 间隔分组，每组内进行插入排序
3. 增量逐步缩小至 1，最终完成排序

#### 核心代码片段

👉 [完整源码](../../src/sorting/02_sequential_list_sorting.cpp)

```cpp
void ShellInsert(SqList &L,int dk)
{
	int i,j;
	for(i=1+dk;i<=L.length ;i++)
	{
		if(L.r[i].key <L.r[i-dk].key )
		{
			L.r[0]=L.r[i];
			for(j=i-dk;j>0 && L.r[j].key >L.r [0].key ;j=j-dk)
			{
				L.r[j+dk]=L.r[j];
			}
			L.r[j+dk]=L.r[0];
		}
	}
}

void ShellSort(SqList &L,int dt[],int t)
{
	int i;
	for(i=0;i<t;i++)
	{
		ShellInsert(L,dt[i]);
	}
}
```

#### 时间/空间复杂度分析

| 指标 | 值 |
|------|-----|
| 平均时间复杂度 | O(n^1.3)（依赖增量序列） |
| 最坏时间复杂度 | O(n²)（Shell 原始增量序列） |
| 最好时间复杂度 | O(n) |
| 空间复杂度 | O(1) |
| 稳定性 | 不稳定 |

增量序列对性能影响极大：
- Shell 原始序列 {1,2,4,8,...}：最坏 O(n²)
- Hibbard 序列 {1,3,7,15,...}：最坏 O(n^1.5)
- Sedgewick 序列 {1,5,19,41,109,...}：最坏 O(n^1.33)

#### 与 C++ STL 标准库实现的对比

| 维度 | 本项目实现 | C++ STL |
|------|-----------|---------|
| 算法 | 希尔排序 | `std::sort` 不使用希尔排序，采用 IntroSort |
| 增量序列 | 硬编码 {5,3,1} | 不适用 |
| 实际地位 | 教学演示 | 希尔排序在现代 C++ 标准库中未被采用，因 IntroSort 在最坏情况下更优 |

#### 已知变体和演进

- **Hibbard 增量**：2^k - 1，最坏 O(n^1.5)
- **Sedgewick 增量**：4^k + 3·2^(k-1) + 1，最坏 O(n^1.33)，实践中最优
- **Pratt 增量**：3-smooth 数列，最坏 O(n log²n) 但常数较大
- **Ciura 增量**：经验最优序列 {1,4,10,23,57,132,301,701}，无严格理论证明

#### 版本/平台相关注意事项

- 本项目增量序列为硬编码，实际应用中应根据数据规模动态选择
- 希尔排序不稳定：相同关键字的记录可能因分组不同而改变相对顺序
- C++ 标准库未提供希尔排序接口

#### Python 实现

👉 [Python 实现](../../python/algorithms/sorting/L02_shell_sort.py)

```python
def shell_sort(arr: list[int], gaps: list[int] | None = None) -> list[int]:
    a = arr[:]
    n = len(a)
    if gaps is None:
        gaps = []
        g = n // 2
        while g > 0:
            gaps.append(g)
            g //= 2
    for dk in gaps:
        for i in range(dk, n):
            if a[i] < a[i - dk]:
                temp = a[i]
                j = i - dk
                while j >= 0 and a[j] > temp:
                    a[j + dk] = a[j]
                    j -= dk
                a[j + dk] = temp
    return a
```

**与 C/C++ 实现差异**：Python 版本默认使用 Shell 增量序列（n//2 递减），支持自定义增量序列参数，无需硬编码增量数组。

```python
    a = arr[:]
    n = len(a)
```

```cpp
int Partition(SqList &L,int low,int high)
{
	L.r[0]=L.r[low];
	int pivotkey=L.r[low].key;
	while(low<high)
	{
		while(low<high&&L.r[high].key>=pivotkey) --high;
		L.r[low]=L.r[high];
		while(low<high&&L.r[low].key<=pivotkey) ++low;
		L.r[high]=L.r[low];
	}
	L.r[low]=L.r[0];
	return low;
}

void QSort(SqList &L,int low,int high)
{
	if(low<high)
	{
		int pivotloc=Partition(L,low,high);
		QSort(L,low,pivotloc-1);
		QSort(L,pivotloc+1,high);
	}
}

void QuickSort(SqList &L)
{
	QSort(L,1,L.length);
}
```

```cpp
void HeapAdjust(SqList &L,int s,int m)
{
	RedType rc=L.r[s];
	for(int j=2*s;j<=m;j*=2)
	{
		if(j<m&&L.r[j].key<L.r[j+1].key) ++j;
		if(rc.key>=L.r[j].key) break;
		L.r[s]=L.r[j];s=j;
	}
	L.r[s]=rc;
}

void CreatHeap(SqList &L)
{
	int n=L.length;
	for(int i=n/2;i>0;--i)
		HeapAdjust(L,i,n);
}

{

	for(int i=L.length;i>1;--i)
```

```python
def _sift_down(arr: list[int], start: int, end: int) -> None:
    root = start
    while True:
        child = 2 * root + 1
        if child > end:
            break
        if child + 1 <= end and arr[child] < arr[child + 1]:
            child += 1
        if arr[root] >= arr[child]:
            break
        arr[root], arr[child] = arr[child], arr[root]
        root = child

def heap_sort(arr: list[int]) -> list[int]:
    a = arr[:]
    n = len(a)
    if n <= 1:
        return a
    for i in range(n // 2 - 1, -1, -1):
        _sift_down(a, i, n - 1)
    for i in range(n - 1, 0, -1):
        a[0], a[i] = a[i], a[0]
        _sift_down(a, 0, i - 1)
    return a
```

```cpp
void Merge(RedType R[],RedType T[],int low,int mid,int high)
{
	int i=low,j=mid+1,k=low;
	while(i<=mid&&j<=high)
	{
		if(R[i].key<=R[j].key) T[k++]=R[i++];
		else T[k++]=R[j++];
	}
	while(i<=mid) T[k++]=R[i++];
	while(j<=high) T[k++]=R[j++];
}

void MSort(RedType R[],RedType T[],int low,int high)
{
	int mid;
	RedType *TR2=new RedType[high-low+1];
	if(low==high) T[low]=R[low];
	else{
		mid=(low+high)/2;
		MSort(R,TR2,low,mid);
		MSort(R,TR2,mid+1,high);
		Merge(TR2,T,low,mid,high);
	}
	delete[] TR2;
}
```

```python
def _merge(left: list[int], right: list[int]) -> list[int]:
    result = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    result.extend(left[i:])
    result.extend(right[j:])
    return result

def merge_sort(arr: list[int]) -> list[int]:
    if len(arr) <= 1:
        return arr[:]
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    return _merge(left, right)
```

```python
        return []

    min_val = min(arr)
```

```cpp
void countingSortByDigit(vector<int>& arr, int exp) {
    int n = static_cast<int>(arr.size());
    vector<int> output(n);
    vector<int> count(10, 0);

    for (int i = 0; i < n; i++) {
        count[(arr[i] / exp) % 10]++;
    }

    for (int i = 1; i < 10; i++) {
        count[i] += count[i - 1];

    for (int i = n - 1; i >= 0; i--) {
        int digit = (arr[i] / exp) % 10;
        output[count[digit] - 1] = arr[i];
        count[digit]--;
    }

}
```

```cpp
    vector<vector<double>> buckets(n);

    for (int i = 0; i < n; i++) {
        int bucketIndex = static_cast<int>(n * arr[i]);
        if (bucketIndex >= n) bucketIndex = n - 1;
        buckets[bucketIndex].push_back(arr[i]);
    }

    for (int i = 0; i < n; i++) {
        insertionSort(buckets[i]);
    }

    int index = 0;
    for (int i = 0; i < n; i++) {
        for (double val : buckets[i]) {
            arr[index++] = val;
        }
    }
}
```

| 空间复杂度 | O(n+k) |
| 稳定性 | 稳定（桶内排序稳定时） |

```python
def bucket_sort(arr: list[float]) -> list[float]:
    n = len(arr)
    if n <= 1:
        return arr[:]
    buckets: list[list[float]] = [[] for _ in range(n)]
    for x in arr:
        bucket_index = int(n * x)
        if bucket_index >= n:
            bucket_index = n - 1
        buckets[bucket_index].append(x)
    for i in range(n):
        buckets[i] = _insertion_sort(buckets[i])
    result: list[float] = []
    for bucket in buckets:
        result.extend(bucket)
    return result
```

#### 概念说明

#### 实现原理

4. 重复直到找到目标或区间为空

```cpp
SearchResult binarySearch(const vector<int>& arr, int target) {
    SearchResult result;
    int low = 0, high = static_cast<int>(arr.size()) - 1;
```

### 插值查找

#### 概念说明

插值查找是二分查找的改进，根据目标值在当前区间的相对位置来估计其索引，而非简单取中点。适用于数据分布均匀的有序数组。

- Weiss, *Data Structures and Algorithm Analysis in C++*, 第 2.6 节

#### 学习前置知识

- 二分查找

#### 实现原理

插值公式：`pos = low + (target - arr[low]) / (arr[high] - arr[low]) * (high - low)`

当数据均匀分布时，该公式能更精确地估计目标位置，减少比较次数。

#### 核心代码片段

👉 [完整源码](../../src/searching/03_interpolation_search.cpp)

```cpp
SearchResult interpolationSearch(const vector<int>& arr, int target) {
    SearchResult result;
    int low = 0, high = static_cast<int>(arr.size()) - 1;

    while (low <= high && target >= arr[low] && target <= arr[high]) {
        result.comparisons++;
        if (arr[high] == arr[low]) {
            if (arr[low] == target) {
                result.found = true;
                result.index = low;
            }
            return result;
        }

        int pos = low + static_cast<int>(
            (static_cast<double>(target - arr[low]) / (arr[high] - arr[low])) * (high - low)
        );

        if (pos < low) pos = low;
        if (pos > high) pos = high;

        result.comparisons++;
        if (arr[pos] == target) {
            result.found = true;
            result.index = pos;
            return result;
        } else if (arr[pos] < target) {
            low = pos + 1;
        } else {
            high = pos - 1;
        }
    }
    return result;
}
```

#### 实现原理

1. 找到大于等于 n 的最小斐波那契数 fibM
2. 用 fibM 的前两个斐波那契数 fib1、fib2 来确定分割点
3. 比较分割点与目标值，根据结果调整斐波那契数和偏移量
4. 重复直到找到目标或区间为空

#### 核心代码片段

```cpp
SearchResult fibonacciSearch(const vector<int>& arr, int target) {
    SearchResult result;
    int n = static_cast<int>(arr.size());
    if (n == 0) return result;

    int fib2 = 0;
    int fib1 = 1;
    int fibM = fib2 + fib1;

    while (fibM < n) {
        fib2 = fib1;
        fib1 = fibM;
        fibM = fib2 + fib1;
    }

    int offset = -1;

    while (fibM > 1) {
        int i = min(offset + fib2, n - 1);

        result.comparisons++;
        if (arr[i] < target) {
            fibM = fib1;
            fib1 = fib2;
            fib2 = fibM - fib1;
            offset = i;
        } else if (arr[i] > target) {
            result.comparisons++;
            fibM = fib2;
            fib1 = fib1 - fib2;
            fib2 = fibM - fib1;
        } else {
            result.comparisons++;
            result.found = true;
            result.index = i;
            return result;
        }
    }

    if (fib1 == 1 && offset + 1 < n) {
        result.comparisons++;
        if (arr[offset + 1] == target) {
            result.found = true;
            result.index = offset + 1;
        }
    }
    return result;
}
```

```python
def fibonacci_search(arr: list[int], target: int) -> int:
    n = len(arr)
    if n == 0:
        return -1
    fib2, fib1 = 0, 1
    fib_m = fib2 + fib1
    while fib_m < n:
        fib2, fib1 = fib1, fib_m
        fib_m = fib2 + fib1
    offset = -1
    while fib_m > 1:
        i = min(offset + fib2, n - 1)
        if arr[i] < target:
            fib_m, fib1, fib2 = fib1, fib2, fib_m - fib1
            offset = i
        elif arr[i] > target:
            fib_m, fib1, fib2 = fib2, fib1 - fib2, fib_m - fib1
        else:
            return i
    if fib1 == 1 and offset + 1 < n and arr[offset + 1] == target:
        return offset + 1
    return -1
```

**与 C/C++ 实现差异**：Python 版本返回索引而非 SearchResult 结构体，使用元组赋值简化斐波那契数更新。

### 拓扑排序

#### 概念说明

拓扑排序是对有向无环图（DAG）的顶点进行线性排序，使得对于每条有向边 (u, v)，u 在排序中出现在 v 之前。本项目使用 Kahn 算法（基于入度的 BFS 方法）实现拓扑排序，同时可检测图中是否存在环。

#### 实现原理

Kahn 算法的核心思想是反复选择入度为 0 的顶点输出，并将其从图中删除（即将其所有邻接顶点的入度减 1）。具体步骤：1) 计算所有顶点的入度；2) 将入度为 0 的顶点入队；3) 循环取出队首顶点，加入排序结果，将其所有邻接顶点的入度减 1，若入度变为 0 则入队；4) 若排序结果包含所有顶点则成功，否则图中存在环。

```cpp
std::vector<int> topologicalSort(int n, const std::vector<std::vector<int>>& adj) {
    std::vector<int> inDegree(n, 0);
    for (int u = 0; u < n; ++u) {
        for (int v : adj[u]) {
            ++inDegree[v];
        }
    }

    std::queue<int> q;
    for (int i = 0; i < n; ++i) {
        if (inDegree[i] == 0) {
            q.push(i);
        }
    }

    std::vector<int> order;
    order.reserve(n);
    while (!q.empty()) {
        int u = q.front();
        q.pop();
        order.push_back(u);
        for (int v : adj[u]) {
            if (--inDegree[v] == 0) {
                q.push(v);
            }
        }
    }

    if (static_cast<int>(order.size()) != n) {
        return {};
    }
    return order;
}
```

```python
def topological_sort(n: int, adj: List[List[int]]) -> List[int]:
    in_degree: List[int] = [0] * n
    for u in range(n):
        for v in adj[u]:
            in_degree[v] += 1

    q: deque[int] = deque()
    for i in range(n):
        if in_degree[i] == 0:
            q.append(i)

    order: List[int] = []
    while q:
        u = q.popleft()
        order.append(u)
        for v in adj[u]:
            in_degree[v] -= 1
            if in_degree[v] == 0:
                q.append(v)

    if len(order) != n:
        return []
    return order
```

### Dijkstra 最短路径

#### 概念说明

Dijkstra 算法求解单源最短路径问题，要求图中所有边权非负。算法采用贪心策略，每次从未确定最短路径的顶点中选择距离最小的顶点，通过该顶点松弛其邻接边。本项目使用最小堆（`std::priority_queue`）优化，时间复杂度为 O((V+E)logV)。

#### 权威参考来源

- Edsger W. Dijkstra, "A Note on Two Problems in Connexion with Graphs", *Numerische Mathematik*, 1959 — Dijkstra 算法原始论文
- 《算法导论》第 24.3 节 "Dijkstra's algorithm"

#### 学习前置知识

- 图的邻接表表示
- 优先队列（最小堆）
- 贪心算法思想
- 松弛操作概念

#### 实现原理

Dijkstra 算法维护一个距离数组 `dist[]`，初始时 `dist[source] = 0`，其余为无穷大。使用最小堆存储 `(距离, 顶点)` 对，每次取出距离最小的顶点 u，若该距离大于 `dist[u]` 则跳过（延迟删除）。否则遍历 u 的所有邻接边 (u, v, w)，若 `dist[u] + w < dist[v]` 则更新 `dist[v]` 并将新距离入堆。算法结束时 `dist[v]` 即为源点到 v 的最短距离。


👉 [完整源码](../../src/graph/dijkstra/25_dijkstra.cpp)

```cpp
std::vector<int> dijkstra(int n, const std::vector<std::vector<std::pair<int,int>>>& adj, int source) {
    std::vector<int> dist(n, INT_MAX);
    dist[source] = 0;

    std::priority_queue<std::pair<int,int>, std::vector<std::pair<int,int>>, std::greater<>> pq;
    pq.push({0, source});

    while (!pq.empty()) {
        auto [d, u] = pq.top();
        pq.pop();

        if (d > dist[u]) {
            continue;
        }

        for (auto [v, w] : adj[u]) {
            if (dist[u] != INT_MAX && dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;
                pq.push({dist[v], v});
            }
        }
    }

    return dist;
}
```

```python
def dijkstra(n: int, adj: List[List[Tuple[int, int]]], source: int = 0) -> List[int]:
    dist: List[int] = [float("inf")] * n
    dist[source] = 0
    pq: List[Tuple[int, int]] = [(0, source)]

    while pq:
        d, u = heapq.heappop(pq)
        if d > dist[u]:
            continue
        for v, w in adj[u]:
            if dist[u] + w < dist[v]:
                dist[v] = dist[u] + w
                heapq.heappush(pq, (dist[v], v))

    return dist
```

### Bellman-Ford

#### 概念说明

Bellman-Ford 算法求解单源最短路径问题，支持负权边，并可检测负权环。算法通过对所有边进行 V-1 轮松弛操作，确保找到最短路径。若第 V 轮仍有更新，则说明存在负权环。时间复杂度 O(VE)。

- 《算法导论》第 24.1 节 "The Bellman-Ford algorithm"

- Ford, L. R., Jr., *Network Flow Theory*, RAND Corporation, 1956

#### 学习前置知识

- 图的边集表示
- 松弛操作
- 负权边与负权环的概念

#### 实现原理

Bellman-Ford 算法对所有边进行 V-1 轮松弛。每轮松弛遍历所有边 (u, v, w)，若 `dist[u] + w < dist[v]` 则更新 `dist[v]`。V-1 轮后，若再遍历所有边仍存在可松弛的边，则图中存在负权环，返回空数组。优化：若某轮无更新则提前终止。


```cpp
std::vector<int> bellmanFord(int n, const std::vector<std::tuple<int,int,int>>& edges, int source) {
    std::vector<int> dist(n, INT_MAX);
    dist[source] = 0;

    for (int i = 0; i < n - 1; ++i) {
        bool updated = false;
        for (auto [u, v, w] : edges) {
            if (dist[u] != INT_MAX && dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;
                updated = true;
            }
        }
        if (!updated) {
            break;
        }
    }

    for (auto [u, v, w] : edges) {
        if (dist[u] != INT_MAX && dist[u] + w < dist[v]) {
            return {};
        }
    }

    return dist;
}
```

```python
def bellman_ford(n: int, edges: List[Tuple[int, int, int]], source: int = 0) -> Optional[List[int]]:
```

### Floyd-Warshall

#### 概念说明

Floyd-Warshall 算法求解所有顶点对之间的最短路径问题，支持负权边（但不可有负权环）。算法通过三重循环，逐步考虑每个顶点作为中转点，更新任意两点间的最短距离。时间复杂度 O(V³)，空间复杂度 O(V²)。

- Warshall, S., "A Theorem on Boolean Matrices", *Journal of the ACM*, 1962

#### 学习前置知识

- 图的邻接矩阵表示

#### 实现原理

Floyd-Warshall 本质是动态规划：`dp[k][i][j]` 表示仅经过顶点 0~k 作为中转时，i 到 j 的最短距离。状态转移：`dp[k][i][j] = min(dp[k-1][i][j], dp[k-1][i][k] + dp[k-1][k][j])`。空间优化后使用二维数组，外层循环枚举中转点 k，内层双重循环枚举顶点对 (i, j)。若对角线出现负值则存在负权环。

```mermaid
    F -->|否| M["k++"]
```

```cpp
std::vector<std::vector<int>> floydWarshall(int n, const std::vector<std::vector<int>>& dist) {
    std::vector<std::vector<int>> d = dist;

    for (int k = 0; k < n; ++k) {
        for (int i = 0; i < n; ++i) {
            for (int j = 0; j < n; ++j) {
                if (d[i][k] < INT_MAX && d[k][j] < INT_MAX) {
                    d[i][j] = std::min(d[i][j], d[i][k] + d[k][j]);
                }
            }
        }
    }

    return d;
}
```

- **路径重建**：额外维护 `next[i][j]` 数组记录中转点，可输出具体路径
- **负权环检测**：算法结束后检查对角线 `d[i][i] < 0`
- **传递闭包**：将 `min` 和 `+` 替换为 `or` 和 `and`，可求传递闭包（Warshall 算法）
- **并行化**：外层循环 k 不可并行，但内层 (i, j) 可并行处理

#### 版本/平台相关注意事项

- 本项目使用邻接矩阵（二维 `vector`）表示图
- `INT_MAX` 加法溢出：代码中通过 `d[i][k] < INT_MAX && d[k][j] < INT_MAX` 前置判断避免
- V 较大时 O(V³) 时间和 O(V²) 空间可能成为瓶颈，建议 V < 500

#### Python 实现

👉 [Python 实现](../../python/algorithms/graph/L27_floyd_warshall.py)

```python
def floyd_warshall(n: int, dist: List[List[int]]) -> List[List[int]]:
    d = [row[:] for row in dist]

    for k in range(n):
        for i in range(n):
            for j in range(n):
                if d[i][k] + d[k][j] < d[i][j]:
                    d[i][j] = d[i][k] + d[k][j]

    return d
```

**与 C/C++ 实现差异**：Python 版本使用 `float("inf")` 替代 `INT_MAX`，加法不会溢出；使用列表推导 `[row[:] for row in dist]` 深拷贝距离矩阵。

### Kruskal 最小生成树

#### 概念说明

Kruskal 算法求解最小生成树问题，从边出发，按权值从小到大排序，逐步加入不构成环的边，直到生成树包含 V-1 条边。使用并查集（Union-Find）检测环，时间复杂度 O(ElogE)，适合稀疏图。

#### 学习前置知识

- 并查集（路径压缩 + 按秩合并）
- 边排序

#### 实现原理

Kruskal 算法的核心是贪心 + 并查集。1) 将所有边按权值从小到大排序；2) 依次取出每条边 (u, v, w)，若 u 和 v 不在同一连通分量（`find(u) != find(v)`），则将该边加入生成树，并合并 u 和 v 所在的连通分量（`unite(u, v)`）；3) 当已选边数达到 V-1 时算法结束。并查集使用路径压缩和按秩合并，均摊复杂度 O(α(V))。

```cpp
int kruskal(int n, const std::vector<std::tuple<int,int,int>>& edges) {
    auto sortedEdges = edges;
    std::sort(sortedEdges.begin(), sortedEdges.end(),
              [](const auto& a, const auto& b) { return std::get<2>(a) < std::get<2>(b); });

    UnionFind uf(n);
    int totalWeight = 0;
    int edgeCount = 0;

    for (auto [u, v, w] : sortedEdges) {
        if (uf.unite(u, v)) {
            totalWeight += w;
            ++edgeCount;
            if (edgeCount == n - 1) {
                break;
            }
        }
    }

    return totalWeight;
}
```

#### 实现原理

```cpp
int prim(int n, const std::vector<std::vector<std::pair<int,int>>>& adj, int start) {
    std::vector<bool> visited(n, false);
    std::priority_queue<std::pair<int,int>, std::vector<std::pair<int,int>>, std::greater<>> pq;

    visited[start] = true;
    for (auto [v, w] : adj[start]) {
        pq.push({w, v});
    }

    int totalWeight = 0;
    int edgeCount = 0;

    while (!pq.empty() && edgeCount < n - 1) {
        auto [w, u] = pq.top();
        pq.pop();

        if (visited[u]) {
            continue;
        }

        visited[u] = true;
        totalWeight += w;
        ++edgeCount;

        for (auto [v, weight] : adj[u]) {
            if (!visited[v]) {
                pq.push({weight, v});
            }
        }
    }

    return totalWeight;
}
```

```python
def prim(n: int, adj: List[List[Tuple[int, int]]], start: int = 0) -> int:
    visited: List[bool] = [False] * n
    pq: List[Tuple[int, int]] = []

    visited[start] = True
    for v, w in adj[start]:
        heapq.heappush(pq, (w, v))

    total_weight = 0
    edge_count = 0

    while pq and edge_count < n - 1:
        w, u = heapq.heappop(pq)
        if visited[u]:
            continue
        visited[u] = True
        total_weight += w
        edge_count += 1
        for v, weight in adj[u]:
            if not visited[v]:
                heapq.heappush(pq, (weight, v))

    return total_weight
```

```cpp
std::vector<MatchResult> bruteForceMatch(const std::string& text, const std::string& pattern) {
    std::vector<MatchResult> results;
    int n = text.length();
    int m = pattern.length();
    int comparisons = 0;

    for (int i = 0; i <= n - m; i++) {
        int j = 0;
        while (j < m) {
            comparisons++;
            if (text[i + j] != pattern[j]) {
                break;
            }
            j++;
        }
        if (j == m) {
            results.push_back({i, comparisons});
        }
    }

    if (results.empty()) {
        results.push_back({-1, comparisons});
    }
    return results;
}
```

```python
def brute_force_search(text: str, pattern: str) -> List[int]:
```

```cpp
    int m = pattern.length();
    std::vector<int> next(m, 0);
```

```python
            j = next_arr[m]
    return positions
```

**与 C/C++ 实现差异**：Python 版本使用 next_arr 替代 next（避免与内置函数冲突），返回所有匹配位置。

### Rabin-Karp 算法

#### 概念说明

Rabin-Karp 算法使用哈希函数对模式串和主串的子串进行快速比较，仅当哈希值匹配时才进行逐字符验证。

#### 权威参考来源

- Richard M. Karp, Michael O. Rabin, "Efficient Randomized Pattern-Matching Algorithms", *IBM Journal of Research and Development*, 1987
- 《算法导论》第 32 章，32.2 节 "The Rabin-Karp algorithm"

#### 学习前置知识

- 哈希函数
- 模运算
- 滚动哈希

#### 实现原理

1. 计算模式串的哈希值和主串第一个窗口的哈希值
2. 滑动窗口时，利用滚动哈希公式 O(1) 更新哈希值
3. 哈希值匹配时进行逐字符验证，避免哈希碰撞导致的误判

👉 [完整源码](../../src/string_algorithm/30_rabin_karp.cpp)

```cpp
std::vector<int> rabinKarpSearch(const std::string& text, const std::string& pattern) {
    std::vector<int> positions;
    int n = text.length();
    int m = pattern.length();

    if (m > n || m == 0) return positions;

    long long h = 1;
    for (int i = 0; i < m - 1; i++) {
        h = (h * BASE) % MOD;
    }

    long long patternHash = 0;
    long long textHash = 0;
    for (int i = 0; i < m; i++) {
        patternHash = (patternHash * BASE + pattern[i]) % MOD;
        textHash = (textHash * BASE + text[i]) % MOD;
    }

    for (int i = 0; i <= n - m; i++) {
        if (patternHash == textHash) {
            bool match = true;
            for (int j = 0; j < m; j++) {
                if (text[i + j] != pattern[j]) {
                    match = false;
                    break;
                }
            }
            if (match) {
                positions.push_back(i);
            }
        }

        if (i < n - m) {
            textHash = (textHash - text[i] * h % MOD + MOD) % MOD;
            textHash = (textHash * BASE + text[i + m]) % MOD;
        }
    }

    return positions;
}
```

#### 概念说明

Boyer-Moore 算法从模式串末尾向前匹配，利用坏字符规则跳过不必要的比较。本项目实现了简化版（仅坏字符规则）。

#### 权威参考来源

- Robert S. Boyer, J Strother Moore, "A Fast String Searching Algorithm", *Communications of the ACM*, Vol.20, No.10, 1977 — Boyer-Moore 算法的原始论文

#### 学习前置知识

- 从右向左匹配的思想
- 坏字符规则与好后缀规则

#### 实现原理

1. **构建坏字符表**：记录每个字符在模式串中最后出现的位置
2. **从右向左匹配**：模式串与主串对齐后，从模式串末尾开始比较
3. **坏字符规则**：失配时，根据主串中失配字符在模式串中的位置决定滑动距离

👉 [完整源码](../../src/string_algorithm/30_boyer_moore.cpp)

```cpp
std::vector<int> buildBadCharTable(const std::string& pattern) {
    int m = pattern.length();
    std::vector<int> badChar(ALPHABET_SIZE, -1);

    for (int i = 0; i < m; i++) {
        badChar[(unsigned char)pattern[i]] = i;
    }

    return badChar;
}

std::vector<int> boyerMooreSearch(const std::string& text, const std::string& pattern) {
    std::vector<int> positions;
    int n = text.length();
    int m = pattern.length();

    if (m == 0 || m > n) return positions;

    std::vector<int> badChar = buildBadCharTable(pattern);

    int s = 0;
    while (s <= n - m) {
        int j = m - 1;

        while (j >= 0 && pattern[j] == text[s + j]) {
            j--;
        }

        if (j < 0) {
            positions.push_back(s);
            int shift = (s + m < n) ? m - badChar[(unsigned char)text[s + m]] : 1;
            s += (shift > 0) ? shift : 1;
        } else {
            int shift = j - badChar[(unsigned char)text[s + j]];
            s += (shift > 0) ? shift : 1;
        }
    }

    return positions;
}
```

| 最坏时间复杂度 | O(nm)（仅坏字符规则时） |
| 预处理时间 | O(m + k) |
| 空间复杂度 | O(k) |

其中 k 为字符集大小。完整 Boyer-Moore（含好后缀规则）最坏 O(n)。

- **Boyer-Moore-Horspool**（1980）：简化版 BM，仅使用坏字符规则中最后一个字符，实现更简单

### 滚动哈希

#### 概念说明

滚动哈希（Rolling Hash）允许在 O(1) 时间内计算滑动窗口的哈希值，是 Rabin-Karp 算法的核心数据结构。

#### 权威参考来源

- Karp, Rabin (1987)，滚动哈希作为 Rabin-Karp 算法的核心组件
- 多项式滚动哈希是应用最广泛的滚动哈希方案

#### 学习前置知识

- 哈希函数
- 模运算
- 多项式求值

#### 实现原理

1. 使用多项式哈希：`H = s[0]*BASE^(m-1) + s[1]*BASE^(m-2) + ... + s[m-1]`
2. 滑动窗口时：`H' = (H - s[out]*BASE^(m-1)) * BASE + s[in]`
3. 所有运算对 MOD 取模，防止溢出

#### 核心代码片段

👉 [完整源码](../../src/string_algorithm/30_rolling_hash.cpp)

```cpp
class RollingHash {
private:
    long long hash;
    long long power;
    int windowSize;

public:
    RollingHash() : hash(0), power(1), windowSize(0) {}

    long long computeHash(const std::string& s) {
        int n = s.length();
        hash = 0;
        power = 1;
        windowSize = n;

        for (int i = 0; i < n; i++) {
            hash = (hash * BASE + s[i]) % MOD;
            if (i < n - 1) {
                power = (power * BASE) % MOD;
            }
        }
        return hash;
    }

    long long roll(char outChar, char inChar) {
        hash = (hash - outChar * power % MOD + MOD) % MOD;
        hash = (hash * BASE + inChar) % MOD;
        return hash;
    }

    long long getHash() const { return hash; }
    int getWindowSize() const { return windowSize; }
};
```

|------|-----|

| 空间复杂度 | O(1) |

### Z 算法（扩展 KMP）

#### 概念说明

Z 算法通过计算 Z 数组实现 O(n) 的字符串预处理和 O(n+m) 的模式匹配。Z 数组定义：Z[i] 表示字符串 S 从位置 i 开始的后缀与 S 本身的最长公共前缀长度。Z 算法是 KMP 算法的替代方案，在某些场景下比 KMP 更直观，且同样支持 O(n+m) 的模式匹配。

#### 权威参考来源

- Dan Gusfield, *Algorithms on Strings, Trees and Sequences*, Cambridge University Press, 1997 — Z 算法的系统阐述
- 《算法导论》第 32 章字符串匹配 — Z 算法与 KMP 的对比

#### 学习前置知识

- KMP 算法（对比理解）
- 字符串前缀与后缀的概念
- 滑动窗口思想

#### 实现原理

Z 算法的核心是维护一个 Z-box [l, r]，表示当前已知的最右匹配区间。对于位置 i：若 i < r，则 Z[i] 至少为 min(r - i, Z[i - l])（利用已知信息避免重复比较）；然后尝试向右扩展 Z[i]。若 i + Z[i] > r，则更新 l = i, r = i + Z[i]。模式匹配时，将模式串与文本串拼接为 `pattern + "$" + text`，Z 数组中值为 m 的位置即为匹配点。

```cpp
std::vector<int> zFunction(const std::string& s) {
    int n = s.length();
    std::vector<int> z(n, 0);
    z[0] = n;
    int l = 0, r = 0;
    for (int i = 1; i < n; i++) {
        if (i < r) {
            z[i] = std::min(r - i, z[i - l]);
        }
        while (i + z[i] < n && s[z[i]] == s[i + z[i]]) {
            z[i]++;
        }
        if (i + z[i] > r) {
            l = i;
            r = i + z[i];
        }
    }
    return z;
}

std::vector<int> zSearch(const std::string& text, const std::string& pattern) {
    std::string combined = pattern + "$" + text;
    std::vector<int> z = zFunction(combined);
    std::vector<int> matches;
    int m = pattern.length();
    for (int i = m + 1; i < (int)z.size(); i++) {
        if (z[i] == m) {
            matches.push_back(i - m - 1);
        }
    }
    return matches;
}
```

```python
def z_function(s: str) -> List[int]:
    n = len(s)
    z: List[int] = [0] * n
    z[0] = n
    l = 0
    r = 0
    for i in range(1, n):
        if i < r:
            z[i] = min(r - i, z[i - l])
        while i + z[i] < n and s[z[i]] == s[i + z[i]]:
            z[i] += 1
        if i + z[i] > r:
            l = i
            r = i + z[i]
    return z

def z_search(text: str, pattern: str) -> List[int]:
    combined = pattern + "$" + text
    z = z_function(combined)
    m = len(pattern)
    return [i - m - 1 for i in range(m + 1, len(z)) if z[i] == m]
```

#### 实现原理

1. **预处理**：在字符间插入 `#`，统一奇偶长度回文处理
2. **维护右边界**：记录当前最右回文边界 right 及其中心 center
3. **利用对称性**：若当前位置在 right 内，利用镜像位置加速计算
4. **中心扩展**：在加速基础上继续尝试扩展回文半径

#### 核心代码片段

👉 [完整源码](../../src/string_algorithm/30_manacher.cpp)

```cpp
std::vector<int> manacher(const std::string& s) {
    std::string t = "#";
    for (char c : s) {
        t += c;
        t += '#';
    }
    int n = t.length();
    std::vector<int> p(n, 0);

    int center = 0, right = 0;

    for (int i = 0; i < n; i++) {
        if (i < right) {
            int mirror = 2 * center - i;
            p[i] = std::min(right - i, p[mirror]);
        }

        while (i - p[i] - 1 >= 0 && i + p[i] + 1 < n && t[i - p[i] - 1] == t[i + p[i] + 1]) {
            p[i]++;
        }

        if (i + p[i] > right) {
            center = i;
            right = i + p[i];
        }
    }

    return p;
}
```

### 最长公共子序列 (LCS)

#### 概念说明

最长公共子序列（LCS）问题求两个字符串的最长公共子序列的长度和内容。子序列不要求连续，但要求保持相对顺序。

- 《算法导论》第 15 章 "Dynamic Programming"，15.4 节 "Longest common subsequence"
- LCS 是动态规划的经典问题之一

#### 学习前置知识

- 动态规划基础
- 二维 DP 表的填写
- 回溯构造解

#### 实现原理

1. 定义 `dp[i][j]` 为 s1 前 i 个字符与 s2 前 j 个字符的 LCS 长度
2. 转移方程：若 `s1[i-1] == s2[j-1]`，则 `dp[i][j] = dp[i-1][j-1] + 1`；否则 `dp[i][j] = max(dp[i-1][j], dp[i][j-1])`
3. 回溯构造 LCS 字符串

```cpp
LCSResult longestCommonSubsequence(const std::string& s1, const std::string& s2) {
    int m = s1.length();
    int n = s2.length();
    std::vector<std::vector<int>> dp(m + 1, std::vector<int>(n + 1, 0));

    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (s1[i - 1] == s2[j - 1]) {
                dp[i][j] = dp[i - 1][j - 1] + 1;
            } else {
                dp[i][j] = std::max(dp[i - 1][j], dp[i][j - 1]);
            }
        }
    }

    std::string lcs;
    int i = m, j = n;
    while (i > 0 && j > 0) {
        if (s1[i - 1] == s2[j - 1]) {
            lcs = s1[i - 1] + lcs;
            i--;
            j--;
        } else if (dp[i - 1][j] > dp[i][j - 1]) {
            i--;
        } else {
            j--;
        }
    }

    return {dp[m][n], lcs};
}
```

> **范式演进逻辑**：
> - **分治 → DP**：当分治的子问题存在大量重叠时（如斐波那契数列），重复计算导致指数级开销。DP 通过记忆化（自顶向下）或制表（自底向上）消除重复计算。

## 动态规划

### 概述

动态规划（Dynamic Programming, DP）是将复杂问题分解为重叠子问题，通过存储子问题的解避免重复计算的方法。

**权威参考来源**：
- Richard Bellman, *Dynamic Programming*, Princeton University Press, 1957 — DP 理论的奠基之作
- 《算法导论》第 14 章 "Dynamic Programming: Rod Cutting" 与第 15 章，系统阐述了 DP 的设计方法

**DP 解题步骤**：

1. **定义状态**：明确 `dp[i]` 或 `dp[i][j]` 的含义
2. **推导转移方程**：找到状态之间的关系
3. **确定边界条件**：初始化基础状态
4. **选择遍历顺序**：确保计算当前状态时依赖的状态已计算

```cpp
long long fib_dp(int n) {
    if (n <= 1) return n;
    vector<long long> dp(n + 1);
    dp[0] = 0;
    dp[1] = 1;
    for (int i = 2; i <= n; i++)

long long fib_matrix(int n) {
    if (n == 0) return 0;
    long long F[2][2] = {{1, 1}, {1, 0}};
    power(F, n - 1);
    return F[0][0];
}
```

```cpp
    if (n <= 2) return n;
```

```cpp
    vector<int> dp(amount + 1, INT_MAX);
    dp[0] = 0;
    for (int i = 1; i <= amount; i++)
        for (int c : coins)
            if (c <= i && dp[i - c] != INT_MAX)
                dp[i] = min(dp[i], dp[i - c] + 1);
    return dp[amount] == INT_MAX ? -1 : dp[amount];
}

int coin_change_combinations(const vector<int>& coins, int amount) {
    vector<int> dp(amount + 1, 0);
    dp[0] = 1;
    for (int c : coins)
        for (int i = c; i <= amount; i++)

}
```

#### 概念说明

编辑距离（Levenshtein Distance）衡量两个字符串的相似度，定义为将一个字符串转换为另一个所需的最少单字符操作（插入、删除、替换）次数。

#### 学习前置知识

- 二维 DP
- 字符串操作

#### 实现原理

- 状态定义：`dp[i][j]` 为 s1 前 i 个字符与 s2 前 j 个字符的编辑距离
- 转移方程：若 `s1[i-1] == s2[j-1]`，`dp[i][j] = dp[i-1][j-1]`；否则 `dp[i][j] = 1 + min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1])`
- 空间优化：仅保留两行，O(min(m,n))

```cpp
int edit_distance_2d(const string& s1, const string& s2) {
    int m = s1.size(), n = s2.size();
    vector<vector<int>> dp(m + 1, vector<int>(n + 1));

    for (int i = 0; i <= m; i++) dp[i][0] = i;
    for (int j = 0; j <= n; j++) dp[0][j] = j;

    for (int i = 1; i <= m; i++)
        for (int j = 1; j <= n; j++) {
            if (s1[i - 1] == s2[j - 1])
                dp[i][j] = dp[i - 1][j - 1];
            else
                dp[i][j] = 1 + min({dp[i - 1][j], dp[i][j - 1], dp[i - 1][j - 1]});
        }
    return dp[m][n];
}
```

```python
    dp: list[list[int]] = [[0] * (n + 1) for _ in range(m + 1)]
```

```cpp
int rob_basic_optimized(const vector<int>& nums) {
    int n = nums.size();
    if (n == 0) return 0;
    if (n == 1) return nums[0];
    int prev2 = nums[0];
    int prev1 = max(nums[0], nums[1]);
    for (int i = 2; i < n; i++) {
        int curr = max(prev1, prev2 + nums[i]);
        prev2 = prev1;
        prev1 = curr;
    }
    return prev1;
}

int rob_circular(const vector<int>& nums) {
    int n = nums.size();
    if (n == 0) return 0;
    if (n == 1) return nums[0];
    if (n == 2) return max(nums[0], nums[1]);

    auto rob_range = [](const vector<int>& arr, int l, int r) -> int {
        int prev2 = arr[l];
        int prev1 = max(arr[l], arr[l + 1]);
        for (int i = l + 2; i <= r; i++) {
            int curr = max(prev1, prev2 + arr[i]);
            prev2 = prev1;
            prev1 = curr;
        }
        return prev1;
    };

    return max(rob_range(nums, 0, n - 2), rob_range(nums, 1, n - 1));
}
```

```cpp
int knapsack_01_2d(int W, const vector<int>& wt, const vector<int>& val, vector<int>& selected) {
    int n = wt.size();
    vector<vector<int>> dp(n + 1, vector<int>(W + 1, 0));

    for (int i = 1; i <= n; i++) {
        for (int w = 0; w <= W; w++) {
            dp[i][w] = dp[i - 1][w];
            if (w >= wt[i - 1])
                dp[i][w] = max(dp[i][w], dp[i - 1][w - wt[i - 1]] + val[i - 1]);
        }
    }

    selected.clear();
    int w = W;
    for (int i = n; i >= 1; i--) {
        if (dp[i][w] != dp[i - 1][w]) {
            selected.push_back(i - 1);
            w -= wt[i - 1];
        }
    }
    reverse(selected.begin(), selected.end());
    return dp[n][W];
}
```

```cpp
int lis_binary(const vector<int>& nums, vector<int>& seq) {
    int n = nums.size();
    if (n == 0) return 0;
    vector<int> tails;
    vector<int> tail_indices;
    vector<int> parent(n, -1);

    for (int i = 0; i < n; i++) {
        auto it = lower_bound(tails.begin(), tails.end(), nums[i]);
        int pos = it - tails.begin();
        if (it == tails.end()) {
            tails.push_back(nums[i]);
            tail_indices.push_back(i);
        } else {
            *it = nums[i];
            tail_indices[pos] = i;
        }
        if (pos > 0)
            parent[i] = tail_indices[pos - 1];
    }

    int len = tails.size();
    seq.clear();
    for (int i = tail_indices[len - 1]; i != -1; i = parent[i])
        seq.push_back(nums[i]);
    reverse(seq.begin(), seq.end());
    return len;
}
```

```python
    if not nums:
        return 0
    tails: list[int] = []
    for num in nums:

        if pos == len(tails):
            tails.append(num)
        else:
            tails[pos] = num
    return len(tails)
```

```cpp
SubarrayResult kadane(const vector<int>& nums) {
    int n = nums.size();
    if (n == 0) return {0, -1, -1};

    int max_sum = nums[0];
    int curr_sum = nums[0];
    int start = 0, end = 0;
    int temp_start = 0;

    for (int i = 1; i < n; i++) {
        if (curr_sum + nums[i] < nums[i]) {
            curr_sum = nums[i];
            temp_start = i;
        } else {
            curr_sum += nums[i];
        }
        if (curr_sum > max_sum) {
            max_sum = curr_sum;
            start = temp_start;
            end = i;
        }
    }
    return {max_sum, start, end};
}
```

#### 概念说明

#### 权威参考来源

- 《算法导论》第 15.2 节 "Matrix-chain multiplication" — 矩阵连乘的权威讲解

#### 学习前置知识

- 矩阵乘法规则（A×B 要求 A 的列数 = B 的行数）
- 动态规划的基本思想
- 区间 DP 的概念

#### 实现原理

状态定义：`dp[i][j]` 表示矩阵链 Aᵢ...Aⱼ 的最少乘法次数。转移方程：`dp[i][j] = min(dp[i][k] + dp[k+1][j] + dims[i]*dims[k+1]*dims[j+1])`，其中 k 为切分点，i ≤ k < j。边界条件：`dp[i][i] = 0`（单个矩阵无需乘法）。遍历顺序：按区间长度 len 从 2 到 n 递增，确保计算 dp[i][j] 时所有子问题已求解。时间复杂度 O(n³)，空间复杂度 O(n²)。

```mermaid
    K --> L["k++"]
    L --> I
    I -->|否| M["i++"]
```

```cpp
int matrixChainOrder(const std::vector<int>& dims) {
    int n = dims.size() - 1;
    std::vector<std::vector<int>> dp(n, std::vector<int>(n, 0));
    for (int len = 2; len <= n; len++) {
        for (int i = 0; i <= n - len; i++) {
            int j = i + len - 1;
            dp[i][j] = INT_MAX;
            for (int k = i; k < j; k++) {
                int cost = dp[i][k] + dp[k+1][j] + dims[i]*dims[k+1]*dims[j+1];
                dp[i][j] = std::min(dp[i][j], cost);
            }
        }
    }
    return dp[0][n-1];
}
```

| 指标 | 值 |
|------|-----|
| 时间复杂度 | O(n³) |
| 空间复杂度 | O(n²) |

> 三重循环：区间长度 O(n)、起始位置 O(n)、切分点 O(n)，因此总时间复杂度为 O(n³)。

#### Python 实现

👉 [Python 实现](../../python/algorithms/dynamic_programming/L31_matrix_chain.py)

```python
def matrix_chain_order(dims: List[int]) -> int:
    n = len(dims) - 1
    if n <= 0:
        return 0
    dp: List[List[int]] = [[0] * n for _ in range(n)]
    for length in range(2, n + 1):
        for i in range(n - length + 1):
            j = i + length - 1
            dp[i][j] = float("inf")
            for k in range(i, j):
                cost = dp[i][k] + dp[k + 1][j] + dims[i] * dims[k + 1] * dims[j + 1]
                if cost < dp[i][j]:
                    dp[i][j] = cost
    return dp[0][n - 1]
```

**与 C/C++ 实现差异**：Python 版本使用 `float("inf")` 替代 `INT_MAX`，不存在溢出问题；使用 `if cost < dp[i][j]` 替代 `std::min` 以便后续扩展记录切分点。

```cpp
double fractional_knapsack(int W, vector<Item>& items) {
    sort(items.begin(), items.end(), [](const Item& a, const Item& b) {
        return a.ratio > b.ratio;
    });

    double total_value = 0.0;
    double remaining = W;

    for (auto& item : items) {
        if (remaining <= 0) break;
        double take = min(item.weight, remaining);
        double take_value = take * item.ratio;
        remaining -= take;
        total_value += take_value;
    }
    return total_value;
}
```

```python
def fractional_knapsack(weights: list[float], values: list[float], capacity: float) -> float:
    items = sorted(
        zip(weights, values),
        key=lambda x: x[1] / x[0],
        reverse=True,
    )
    total_value = 0.0
    remaining = capacity
    for w, v in items:
        if remaining <= 0:
            break
        take = min(w, remaining)
        total_value += take * (v / w)
        remaining -= take
    return total_value
```

```python
            s += nums[i]
```

```cpp
double closestPairDC(vector<Point>& points, int left, int right) {
    if (right - left <= 3) {
        return bruteForce(points, left, right);
    }

    int mid = (left + right) / 2;
    double midX = points[mid].x;

    double dl = closestPairDC(points, left, mid);
    double dr = closestPairDC(points, mid + 1, right);
    double d = min(dl, dr);

    vector<Point> strip;
    for (int i = left; i <= right; i++) {
        if (abs(points[i].x - midX) < d) {
            strip.push_back(points[i]);
        }
    }

    sort(strip.begin(), strip.end(), [](const Point& a, const Point& b) {
        return a.y < b.y;
    });

    for (int i = 0; i < (int)strip.size(); i++) {
        for (int j = i + 1; j < (int)strip.size() && strip[j].y - strip[i].y < d; j++) {
            double dist = distance(strip[i], strip[j]);
            if (dist < d) d = dist;
        }
    }

    return d;
}
```

```cpp
class PermutationsUsed {
    vector<vector<int>> results;
    vector<int> current;
    vector<bool> used;

    void backtrack(const vector<int>& nums) {
        if ((int)current.size() == (int)nums.size()) {
            results.push_back(current);
            return;
        }
        for (int i = 0; i < (int)nums.size(); i++) {
            if (used[i]) continue;
            if (i > 0 && nums[i] == nums[i - 1] && !used[i - 1]) continue;
            used[i] = true;
            current.push_back(nums[i]);
            backtrack(nums);
            current.pop_back();
            used[i] = false;
        }
    }

public:
    vector<vector<int>> solve(vector<int> nums) {
        results.clear();
        current.clear();
        used.assign(nums.size(), false);
        sort(nums.begin(), nums.end());
        backtrack(nums);
        return results;
    }
};
```

```python
def permute(nums: list[int]) -> list[list[int]]:
    results: list[list[int]] = []
    def backtrack(start: int) -> None:
        if start == len(nums):
            results.append(nums[:])
            return
        for i in range(start, len(nums)):
            nums[start], nums[i] = nums[i], nums[start]
            backtrack(start + 1)
            nums[start], nums[i] = nums[i], nums[start]
    backtrack(0)
    return results
```

**与 C/C++ 实现差异**：Python 版本使用交换法+闭包实现，无需类封装和 used 数组。

```cpp
    vector<int> current;

    void backtrack(int n, int k, int start) {
        if ((int)current.size() == k) {
            results.push_back(current);
            return;
```

```python
def combine(n: int, k: int) -> list[list[int]]:
    results: list[list[int]] = []
    current: list[int] = []
    def backtrack(start: int) -> None:
        if len(current) == k:
            results.append(current[:])
            return
        for i in range(start, n - (k - len(current)) + 2):
            current.append(i)
            backtrack(i + 1)
            current.pop()
    backtrack(1)
    return results
```

```cpp
class SubsetSum {
    vector<vector<int>> results;
    vector<int> current;

    void backtrack(const vector<int>& nums, int target, int start, int curr_sum) {
        if (curr_sum == target) {
            results.push_back(current);
            return;
        }
        for (int i = start; i < (int)nums.size(); i++) {
            if (curr_sum + nums[i] > target) break;
            if (i > start && nums[i] == nums[i - 1]) continue;
            current.push_back(nums[i]);
            backtrack(nums, target, i + 1, curr_sum + nums[i]);
            current.pop_back();
        }
    }

public:
    vector<vector<int>> solve(vector<int> nums, int target) {
        results.clear();
        current.clear();
        sort(nums.begin(), nums.end());
        backtrack(nums, target, 0, 0);
        return results;
    }
};
```

```python
def subset_sum(nums: list[int], target: int) -> list[list[int]]:
    results: list[list[int]] = []
    current: list[int] = []
    sorted_nums = sorted(nums)
    def backtrack(start: int, curr_sum: int) -> None:
        if curr_sum == target:
            results.append(current[:])
            return
        for i in range(start, len(sorted_nums)):
            if curr_sum + sorted_nums[i] > target:
                break
            if i > start and sorted_nums[i] == sorted_nums[i - 1]:
                continue
            current.append(sorted_nums[i])
            backtrack(i + 1, curr_sum + sorted_nums[i])
            current.pop()
    backtrack(0, 0)
    return results
```

```cpp
class NQueensBitmask {
    int n;
    int count;

    void backtrack(int row, int cols, int diag1, int diag2) {
        if (row == n) {
            count++;
            return;
        }
        int available = ((1 << n) - 1) & ~(cols | diag1 | diag2);
        while (available) {
            int pos = available & (-available);
            available -= pos;
            backtrack(row + 1, cols | pos, (diag1 | pos) << 1, (diag2 | pos) >> 1);
        }
    }
};
```

```python
                return False
        return True

        if row == n:

            ...
```

#### 实现原理

- **欧几里得算法（GCD）**：`gcd(a, b) = gcd(b, a % b)`，递归至 `b == 0`；扩展欧几里得可求解 `ax + by = gcd(a, b)` 的整数解
- **素数判定**：试除法 `O(√n)`，6k±1 优化；Miller-Rabin 概率性素性测试，基于 Fermat 小定理的逆否命题
- **埃拉托斯特尼筛法**：从 2 开始标记合数，每个素数 p 标记 p², p²+p, p²+2p, …，时间 O(n log log n)
- **快速幂（模幂）**：二进制拆分指数，`base^exp = ∏ base^(2^i)`（exp 的第 i 位为 1 时累乘），O(log exp)
- **矩阵快速幂**：将递推关系表示为矩阵乘法，再用快速幂加速，如 Fibonacci 的 `[[1,1],[1,0]]^n`
- **模逆元**：扩展欧几里得求 `a^(-1) mod m`，要求 `gcd(a, m) == 1`
- **卡特兰数**：递推 `C(n) = Σ C(i)×C(n-1-i)`，动态规划求解
- **组合数**：递推 `C(n,k) = C(n-1,k-1) + C(n-1,k)`，滚动数组优化空间

```cpp
int64_t fibonacciMatrix(int n) {

    if (n == 1) return 1;
    int64_t result[2][2] = {{1, 0}, {0, 1}};
    int64_t base[2][2] = {{1, 1}, {1, 0}};
    int64_t mod = 1000000007;
    int p = n - 1;
    while (p > 0) {
        if (p & 1) matrixMultiply(result, base, mod);
        matrixMultiply(base, base, mod);
        p >>= 1;
```
